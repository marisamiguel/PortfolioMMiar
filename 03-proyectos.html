SERVICE CLOUD · AUTOMATIZACIÓN

Gestión avanzada de casos

Diseñé e implementé una solución completa de atención al cliente sobre Salesforce Service Cloud, centralizando todos los canales de entrada (email, teléfono) en una única cola de trabajo inteligente. Configuré Omnichannel Routing con capacidades y habilidades de agente, definiendo reglas de enrutamiento basadas en prioridad, carga de trabajo y SLA para garantizar que cada caso llegase al agente más adecuado en el menor tiempo posible.

Automaticé la tipificación de casos mediante Flows y reglas de asignación, eliminando la clasificación manual y reduciendo el tiempo de primera respuesta. Incorporé escalado automático por prioridad cuando los tiempos de resolución superaban umbrales definidos, asegurando el cumplimiento de los acuerdos de nivel de servicio.

Complementé la solución con dashboards y reportes en tiempo real para supervisores, incluyendo métricas clave como AHT, FCR y tiempo de espera en cola.

SALES CLOUD · INTEGRACIÓN

Implementación de solución de integración

Desarrollé componentes a medida con Lightning Web Components (LWC) y Aura para extender la interfaz nativa de Salesforce, adaptando la experiencia de usuario a los procesos comerciales específicos del cliente.

Implementé lógica de negocio compleja en Apex, incluyendo triggers, clases de servicio y batch jobs para el procesamiento masivo de datos.

La solución requirió una arquitectura de integración multicapa: conecté Salesforce con ERPs externos como SAP mediante MuleSoft y plataformas middleware.

Diseñé APIs REST y SOAP para intercambio bidireccional de información con bases de datos externas asegurando consistencia, manejo de errores y trazabilidad.

Implementé Named Credentials, Auth Providers, análisis estático con PMD y revisiones CRUD/FLS, siguiendo principios SOLID y governor limits.

Sales Cloud Apex & Triggers LWC & Aura APIs REST / SOAP MuleSoft SQL Server Named Credentials

LOGÍSTICA · SERVICE CLOUD · OMNICHANNEL

Optimización de operaciones para empresa de logística

Lideré el diseño e implementación de una solución integral sobre Salesforce Service Cloud para la gestión de incidencias, entregas fallidas y reclamaciones de cliente.

El principal reto fue unificar teléfono, email, portal web y mensajería en una única plataforma operativa.

Implementé Omnichannel Routing adaptado a incidencias logísticas: pérdida de paquete, retrasos, devoluciones y mercancía prioritaria.

Los Flows automatizaron la clasificación y asignación de casos según zona geográfica, SLA y tipo de servicio.

Integré Sales Cloud para conectar oportunidades comerciales con el histórico operativo del cliente y diseñé dashboards ejecutivos con métricas en tiempo real.

AUTOMOCIÓN · DESARROLLO A MEDIDA · SEGURIDAD

Plataforma Salesforce a medida para el sector de automoción

Participé en la implementación de una solución Salesforce completamente personalizada para procesos de venta, concesionarios y posventa.

Desarrollé interfaces avanzadas con LWC y Aura para configuradores, fichas técnicas dinámicas y módulos de garantías.

Implementé comunicación entre componentes mediante Lightning Message Service y arquitectura reutilizable.

Utilicé Checkmarx para análisis estático de seguridad, corrigiendo vulnerabilidades SOQL Injection, CRUD/FLS, XSS y exposición insegura de datos.

Cada entrega seguía un ciclo de escaneo, revisión de hallazgos y validación antes de producción.

SEGUROS · SERVICE CLOUD · ARQUITECTURA LEGACY

Transformación digital de operaciones en empresa aseguradora

Gestioné de forma autónoma una plataforma Salesforce Service Cloud enterprise en entorno legacy.

Diseñé soluciones para siniestros, pólizas y gestión de incidencias utilizando Apex, LWC, Triggers y automatización mediante Flows.

Implementé Email-to-Case y la integración con call center externo para automatizar la creación de casos.

Trabajé además sobre configuración DNS y autenticación SPF, DKIM y DMARC para garantizar entregabilidad y seguridad del correo.

El proyecto implicó modernizar progresivamente una plataforma crítica sin afectar la continuidad del servicio en producción.
